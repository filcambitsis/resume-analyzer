# Resume Analyzer

A web app that analyzes resumes (text or PDF) and gives feedback based on detected skills

## Features
- Paste your resume or upload a PDF
- Detects skills across different categories (technical, business, management, etc.)
- Calculates a simple score
- Gives feedback and suggestions for improvement
- Shows missing skill areas

## Tech Stack
- Python (Flask)
- HTML & CSS
- PyPDF2

## How to Run
1. Clone the repository
3. Run app.py
4. Open in your browser
